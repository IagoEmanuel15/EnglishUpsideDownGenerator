# English (Upside Down) Generator
The generator was made to facilitate the conversion of letters following the special standards of each English (Upside Down) characters. 
The generator will be updated as new English (Upside Down) translation rules arise in the [Minecraft: Java Edition](https://crowdin.com/project/minecraft) and [Minecraft Launcher](https://crowdin.com/project/minecraft-launcher) project.

The generator follows a strict list of characters already approved within the Minecraft project. See the [documentation](https://docs.google.com/document/d/1LRglnkwFtofV7hKfG29mwMs90GiZVdKg7StQJLq1drw/edit?usp=sharing) that lists each one.
